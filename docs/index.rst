Overview
--------

TAOIST-MC generates realistic intergalactic medium (IGM) transmission curves
for studies of Lyman continuum (LyC) emission from distant galaxies. The package
has been modernized with:

- Modular object-oriented design
- Parallel CPU and GPU acceleration (10×+ speedup)
- YAML-based configuration with Pydantic validation
- Automatic sightline caching and reuse
- Command-line interface via Typer
